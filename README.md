# MyET AI – Personalized Interactive News Experience

## Problem
Business news today is static and not personalized. Users must read multiple articles to understand a single topic.

## Solution
MyET AI transforms news into an interactive and personalized experience using AI.

## Features
- AI-powered news summarization
- Personalized news experience
- Interactive Q&A system

## System Architecture
The system includes:
- User Interface
- Input Layer
- Data Layer
- AI Processing Layer (Summarization, Personalization, Q&A)
- Output Layer

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- AI: Mock AI logic (extendable to LLM APIs)

## How to Run
1. Install Python & Flask
2. Run: python app.py
3. Open index.html in browser

## Impact
- Reduces news reading time by 80%
- Improves understanding
- Provides personalized content
